# StudyBuddy v2 · 封测版本

> 异地（或同地）大学情侣的关系容器。一个 app 装下你们的专注、陪伴、地图和回忆。

---

## 安装

把 `studybuddy-v2-release.apk` 发给你的搭档，两人都装上。

> Android 8.0 (API 26) 及以上。需要授予位置和相册权限。

## 第一次打开

1. **注册账号** —— 邮箱 + 密码（任意邮箱都行，不用激活）
2. **绑定搭档** —— 主页右上角齿轮 → 复制自己的"配对码"，让 TA 输入；或者你输入 TA 的码
3. **起个名字** —— 进入"陪伴 / 宠物"，长按"未命名"给你们的伙伴起名
4. **开始用就好** —— app 会跟着你们一起长大

## 你们能用到的功能

### 🟠 今天 tab
- 快速开始专注（倒计时 / 正计时切换）
- 双人状态卡：你和 TA 今天专注了多少分钟、TA 是否在线
- 多主题铭牌：可以建多个主题（"GRE 单词" / "毕设" / "论文"），每次专注归到某个主题
- 右滑进**话廊**：记录你们的小灵感、引用语
- 4 个连接卡：金库、数据统计、专注历史、设置

### ✉ 写信 tab
- 一封信，慢慢写，给 TA 看
- 支持长文，不限字数
- TA 读过会标"TA 读过了"

### 🐱 陪伴 tab（横滑切换三个子页）
- **宠物**：你们共养的像素生物
  - 8 个品种：鞍部猫 / 暹罗 / 狐狸 / 兔子 / 奶牛 / 柴犬 / 水豚 / 刺猬
  - 4 维属性：饱腹 / 心情 / 清洁 / 亲密度
  - 4 种互动：喂食 / 陪玩 / 清洁 / 抚摸
  - 顶部右上角"图鉴"图标 → 全屏图鉴换品种（有二次确认，进度保留）
  - **会自己呼吸、跳起、开心摇摆、吃东西时弹性张嘴**
  - 自动衰减：饿了 / 脏了 / 心情差 → 主页会有 coral 提示
- **金库**：你们的"承诺账本" + 共同基金
  - 工作日断连自动入金（默认 ¥5）
  - 也可以手动充值 / 取出 / 核销
- **便签墙**：朋友圈式时间流
  - 头像 + 名字 + 时间 + 文字 + 图片（最多 4 张）
  - 长按自己发的便签删除
  - 点图片全屏看（左右滑切换多图）

### 🤝 在哪 tab
- 双方实时位置（500m 网格化精度，保护隐私）
- 距离显示："在一起" / "几百米外" / "多少 km 外"
- 共享地标（你们一起去过的地方）

### ⚙ 设置（主页右上角齿轮）
- **App 模式切换**（重要）：
  - **专注模式**（默认）：宠物会衰减，TA 开始专注会推 banner，工作日断连扣金库
  - **娱乐模式**：宠物不衰减、专注向 banner 静默；其它陪伴功能照常
- 默认专注模式（倒计时/正计时）+ 默认时长
- 主页双人卡样式（波形 / 字号）
- 浅色 / 深色主题
- 金库每份金额（¥3 ~ ¥50）
- **情境提醒**：5 种 banner 单独开关
  - 见面开始 / 见面结束 / 共同停留回家后 / TA 开始专注 / 工作日断连
- **解除搭档关系**：7 天冷静期，期间宠物冬眠，任一方都可反悔

### 💌 情境层 banner（不打扰）
app 会在 Home 顶部偶尔浮一张卡：
- **见面开始** —— 双方距离 < 500m + 持续 60min："拍一张吧"
- **见面结束** —— 距离突变 > 1km："记一下"
- **共同停留** —— 你们一起在某个地标 60min+，回家后才提醒："写一段"
- **TA 开始专注** —— "加入 TA"（直接进同步专注）
- **工作日断连** —— "看金库"（自动扣的钱去哪了）

向下滑掉就关，同种 24h 内不再推。

## 已知限制（封测版）

1. **宠物动作还较少** —— 当前只有跳起 / 喂食 / 开心摇摆 / 呼吸，眨眼和更多互动反馈下一版补
2. **狐狸 / 暹罗等品种的形态差异主要靠色块** —— 后续会画更多专属轮廓
3. **地图共享地标 marker 暂未渲染** —— 数据已存，UI 下一版补
4. **见面记录的历史轨迹未上线** —— 见面被自动记录但还没专门的回顾页
5. **推送只在 app 前台时收到** —— 没有 FCM / 厂商推送，锁屏不会响
6. **图片单张 5MB 上限**，没有客户端压缩，拍照原图可能上传失败
7. **EB Garamond 字体用 fallback** —— 目前是系统 Serif
8. **反向地理编码未启用** —— 地标 / 见面位置没自动命名

- 哪个交互让你"诡异"或"不顺"

## 隐私说明

- 所有数据存在我自建的 PocketBase（catclaw.cloud），不接第三方
- 位置精度 500m 网格化，原始 GPS 不写云
- 解绑后所有数据保留但归档（不自动删除）
- 没有埋点、没有广告、没有商业模式
